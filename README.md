# hello-world

This is a simple Hello World application, to show you how to start writing code in Java.

## Clone the repository
```
$ git clone https://github.com/myjavacoretutorial/hello-world.git
```

## Compile the code

```
$ javac HelloWorld.java
```

## Run the application

```
$ java HelloWorld
Hello World !!!
```
